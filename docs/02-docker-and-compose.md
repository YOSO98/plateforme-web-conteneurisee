# 02 - Docker & Docker Compose

##  Objectif
Conteneuriser une application web et la déployer
dans un environnement Cloud à l’aide de Docker
et Docker Compose.

##  Docker
Une image Docker personnalisée a été créée
à partir d’une image Python officielle.

Fonctionnalités :
- installation des dépendances
- exposition du port applicatif
- variables d’environnement
- exécution de l’application

##  Docker Compose
Docker Compose permet de définir et déployer
l’application de manière déclarative.

Fonctionnalités :
- mapping de ports
- variables d’environnement
- redémarrage automatique
- gestion des logs

##  Tests
L’application a été testée :
- en local (Flask)
- via Docker
- via Docker Compose

##  Analyse
L’utilisation de Docker Compose facilite le
déploiement reproductible de l’application
dans un environnement Cloud.
